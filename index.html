ZC Zero Cap Finance — Analytics Tool

A private creator dashboard for Zero Cap Finance.

Pulls the operator's own TikTok and YouTube performance metrics into a single local dashboard. Built to support content iteration: which hooks retain viewers, which formats convert to follows, which beats lose attention. Not a public service. No third-party data. No advertising.

What it does

The tool authenticates with TikTok and YouTube using the official Display API and Data API respectively, fetches the operator's own creator metrics on a daily schedule, and renders a local dashboard. All data lives on the operator's machine. Nothing is uploaded, shared, sold, or shown to third parties.

Daily metrics aggregation

Subscriber counts, follower deltas, video views, watch-time, and engagement totals roll into a single state file refreshed each morning by a scheduled task.

Per-video performance

Top and bottom performers surfaced for each posting cadence so the operator can identify hook patterns that work and patterns that don't.

Cross-platform view

TikTok and YouTube metrics in one place so the operator can compare format performance across the two short-form surfaces.

Audit trail

Per-day JSON snapshots accumulate over time, enabling trend analysis and regression detection across weeks and months.

TikTok scopes used

Each scope is requested only because a specific dashboard feature depends on it. All access is read-only.

user.info.profile

Read the authenticated creator's profile fields (display name, avatar, bio, follower/following counts) to display channel context inside the dashboard.

user.info.stats

Pull aggregate creator statistics (total followers, total likes, total video count) to compute follower deltas and growth-rate curves over time.

video.list

List the operator's own published videos with engagement metadata (views, likes, comments, shares) so the dashboard can identify top/bottom performers and compute hook-iteration metrics. Only the operator's own videos are queried.

Read-only by design

The application never publishes content, modifies profile data, messages other users, or accesses other creators' information. Tokens can be revoked from TikTok account settings at any time.

Privacy posture

Single-user, local-only. The application runs on the operator's local machine. There is no backend server, no database, no CDN, no analytics pixel, no third-party SDK. All fetched data is stored on disk in a YAML state file and a per-day JSON archive.

No data is shared, sold, or transmitted. The application never sends data outside the local machine, never contacts third-party APIs other than TikTok and YouTube to fetch the operator's own metrics, and never displays any user-facing UI to people other than the operator.

Channel

The Zero Cap Finance brand publishes finance-education shorts on YouTube and TikTok. The dashboard exists to support content iteration on that brand.
